# 图片切片台

一个本地可运行的在线图片裁剪网页，支持：

- 2×2、3×3、4×4、5×5 宫格自动裁剪，也支持自定义行列。
- 鼠标拖动外框边线、角点、内部宫格线来调整输出大小。
- 自由切片模式，可添加多个切片框并拖动、缩放。
- 一次导入多张图片，一键批量导出。
- Chrome、Edge 等支持 File System Access API 的浏览器可直接选择本地文件夹写入；其他浏览器会自动改为逐张下载。

## 使用

在当前目录启动任意本地静态服务器，然后打开 `index.html`。

### 可以直接用浏览器打开index.html

### 如果电脑有 Python：

```powershell
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```
